Turtle

Turtles (черепахи) - образовательные роботы, разработанные в конце 1940-х годов и используемые в обучении информатике и машиностроению.

Range

x = range(6)
for n in x:
    print(n)

Random

Библиотека random в Python даёт различные функции для генерации псевдослучайных чисел и операций. Он обычно используется для таких задач, как моделирование случайности, перетасовка последовательностей, генерация случайных данных, а также реализация игр и симуляций.

import random
for _ in range(10):
    print(random.randint(1, 2))

Пример

import turtle

# Создание экрана
screen = turtle.Screen()
screen.bgcolor("white")

# Создание черепахи
my_turtle = turtle.Turtle()

# Приводим черепаху в движение
for _ in range(3):
    my_turtle.forward(100)
    my_turtle.right(90)

# Остановить программу после закрытия окна
screen.exitonclick()

Команды

my_turtle.forward(100) # идти вперёд 100 шагов
my_turtle.backward(100) # идти назад 100 шагов
my_turtle.left(90) # повернуть влево на 90 градусов
my_turtle.right(90) # повернуть вправо на 90 градусов
my_turtle.color("#ff0000") # черепаха рисует красным
my_turtle.circle(30) # круг с радиусом 30
my_turtle.speed(4) # скорость (0 - максимальная)
my_turtle.pensize(3) # ширина линии, которую оставляет черепаха
my_turtle.setposition(100, 80) # x, y - координаты; (0, 0) координаты центра

# заливка (комбинация команд)
my_turtle.begin_fill()
… # здась рисуем форму, которую будем заполнять
my_turtle.end_fill()

# наша черепаха не будет рисовать при движении
my_turtle.penup()
# наша черепаха будет рисовать при движении
my_turtle.pendown()

my_turtle.goto(40, 30) # перейти в координаты x, y

screen.title("Моя черепаха") # изменить название заголовка окна
my_turtle.shape("turtle") # изменить форму черепахи
my_turtle.fillcolor("green") # изменить цвет черепахи
